# คู่มือการใช้งาน Switch Language

## สารบัญ
1. ข้อมูลเบื้องต้น
2. รายละเอียดฟีเจอร์
3. วิธีการใช้งาน

## ข้อมูลเบื้องต้น 
Switch Language ใช้สำหรับการเปลี่ยนภาษาหน้าเว็บให้แสดงภาษาต่างๆ เช่น ภาษาไทย ภาษาอังกฤษ และภาษาจีน

## รายละเอียดฟีเจอร์
- ทำการเปลี่ยนภาษาหน้าเว็บ
